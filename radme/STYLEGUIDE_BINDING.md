# STYLEGUIDE (Binding)

This document is mandatory for all generated UI code.

## Core
- Colors: use only brand tokens: brand-primary, brand-cta, brand-highlight, brand-bg (no arbitrary colors)
- Fonts: Public = Lato + Playfair Display (headings), Admin = Inter
- Typography: base 16px, minimum 14px

## Components
- Default radius: rounded-2xl
- Buttons: Primary / Secondary / CTA only
- Inputs: rounded-2xl + focus:ring-brand-primary/20
- Cards: bg-white, rounded-2xl, shadow-md

## Layout & Motion
- Container: max-w-7xl centered
- Grid-based layouts
- Transitions: subtle only (duration-300)

## Icons & A11y
- Icons: Material Symbols Outlined only
- WCAG AA contrast, focus-visible required, min touch target 44px
